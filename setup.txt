1. Unzip the .zip file and extract to a directory: C:\minecraftServer
2. Run the C:\minecraftServer\runServer.bat (double click from windows explorer) 
3. Edit eula.txt with a text editor (like notepad) and change eula=false to eula=true
4. Run the C:\minecraftServer\runServer.bat again
5. A Minecraft server should appear, accept the firewall popup if it appears
